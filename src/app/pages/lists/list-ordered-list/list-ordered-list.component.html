Ordered Lists

Ordered lists are lists with numerical content

This is a simple Ordered list

  1. First Item
  2. Second Item
  3. Third Item

However, the HTML specification allows for nesting of these lists

  1. Smith
    1. Stan
    2. Francine
    3. Steve
    4. Hayley
    5. Roger
  2. Simpson
    1. Homer
    2. Marge
    3. Bart
    4. Lisa
    5. Maggie

They even go as far as allowing unordered lists within them

  1. Smith
    • Stan
    • Francine
    • Steve
    • Hayley
    • Roger
  2. Simpson
    • Homer
    • Marge
    • Bart
    • Lisa
    • Maggie
