Radios

The often misunderstood multiple choice component.

So what are the rules for radios:

  1. They must be announced as a group.
  2. The group must be described by "question".
  3. Each radio must describe a unique answer (values can be duplicated)
  4. Each radio must announce it's position in the list (e.g 1 of 4)
  5. Each radio must announce it's selected status

It seems that if radios are off the bottom of the screen that Voiceover on iOS cant access them, as such a button is required off screen.
